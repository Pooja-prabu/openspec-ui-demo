# OpenSpec UI Demo

This project explores **spec-driven software development using OpenSpec**.

Instead of writing code directly, the application structure is defined through specifications which AI tools can interpret to generate software components.

## Features

- Spec-driven React UI architecture
- Component-based UI structure
- State management setup
- Styling and responsive design configuration
- Development and production build pipelines

## Purpose

This project was built during an **AI Workshop on Secure Digital Futures**, where we explored modern AI-assisted development workflows including:

- Agentic AI
- Spec-driven development
- AI coding assistants
- Automated UI generation

## Technologies Used

- OpenSpec
- React
- Node.js
- VS Code
- AI coding assistants
